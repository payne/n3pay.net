Back in January 2026 I was exited about [AtlasOS](https://n3pay.net/blog/AtlasOS/AtlasOS1/).

I use my Windows 11 pro laptop for four types of things:
1. HAM Radio (winLink Express, VARA, etc)
2. Weekly Fortnite games with a best friend
3. WSL running Ubuntu
4. Complaining about Windows 11 pro.

The AtlasOS setup recently prevented 2 and made 1 hard.  So I asked [Claude.AI how to reset](https://claude.ai/share/88a0e46f-f1d1-45f2-8acc-3c9f6065b884), its advice did not work for until it suggested using the Windows 11 ISO:

Repair via Windows 11 ISO

1. Download the Windows 11 ISO from microsoft.com/software-download/windows11
2. Mount it (double-click the ISO)
3. Run setup.exe from the mounted drive and choose Keep my files and apps — this is an in-place repair that restores missing system files without wiping anything
